🚀 如何上传到 Vercel？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

方法 1：通过 GitHub Desktop（最简单）
────────────────────────────────────
1. 下载 GitHub Desktop: https://desktop.github.com/
2. 打开 GitHub Desktop
3. File → Add Local Repository
4. 选择文件夹: c:\Users\Jacks\新建文件夹\pt-assessment-simulator
5. 如果提示 "not a git repository"，点击 "Create a repository"
6. 填写提交信息，点击 "Commit to main"
7. 点击 "Publish repository"
8. 在 Vercel 导入这个 GitHub 仓库

方法 2：通过命令行（如果安装了 Git）
────────────────────────────────────
cd c:\Users\Jacks\新建文件夹\pt-assessment-simulator
git init
git add .
git commit -m "Initial commit: PT Assessment Simulator"
# 在 GitHub 创建仓库后：
git remote add origin https://github.com/YOUR_USERNAME/pt-assessment-simulator.git
git branch -M main
git push -u origin main

方法 3：直接在 Vercel 上传（无需 GitHub）
────────────────────────────────────
1. 压缩整个 pt-assessment-simulator 文件夹为 .zip
2. 访问 https://vercel.com
3. 点击 "Add New" → "Project"
4. 选择 "Import from ZIP"
5. 上传 zip 文件
